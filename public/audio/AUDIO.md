# Audio Files Guide

This directory contains all audio files for the Persona-themed portfolio, organized by theme.

## Folder Structure

```
audio/
├── p3/                    # Persona 3 theme audio
│   ├── menu-bgm.mp3      # Background music for main menu
│   ├── portfolio-bgm.mp3 # Background music for portfolio page
│   ├── hover.mp3         # Sound effect for hover interactions
│   └── select.mp3        # Sound effect for click/select
├── p4/                    # Persona 4 theme audio
│   ├── menu-bgm.mp3
│   ├── portfolio-bgm.mp3
│   ├── hover.mp3
│   └── select.mp3
└── p5/                    # Persona 5 theme audio
    ├── menu-bgm.mp3
    ├── portfolio-bgm.mp3
    ├── hover.mp3
    └── select.mp3
```

## File Requirements

### Background Music (BGM)
- **Format:** MP3
- **Bitrate:** 128-192 kbps recommended
- **Loop:** Should be seamlessly loopable (start and end should connect smoothly)
- **Duration:** 1-3 minutes recommended
- **File names:**
  - `menu-bgm.mp3` - Plays on the main menu screen
  - `portfolio-bgm.mp3` - Plays on the portfolio page

### Sound Effects (SFX)
- **Format:** MP3
- **Bitrate:** 128 kbps recommended
- **Duration:** < 1 second (short and snappy)
- **File names:**
  - `hover.mp3` - Plays when hovering over buttons/links
  - `select.mp3` - Plays when clicking buttons/links

## How to Replace Audio

1. **Prepare your audio files** in MP3 format
2. **Name them exactly** as specified above
3. **Place them** in the corresponding theme folder:
   - For Persona 3 theme → `p3/` folder
   - For Persona 4 theme → `p4/` folder
   - For Persona 5 theme → `p5/` folder

## Tips for Best Results

### Background Music
- Use royalty-free music or music you have rights to use
- Ensure the loop point is seamless (no audio gap or pop)
- Consider the theme vibe:
  - **P3:** Melancholic, mysterious (blue theme)
  - **P4:** Upbeat, energetic (yellow theme)
  - **P5:** Stylish, jazzy (red/black theme)

### Sound Effects
- Keep them short and responsive
- Match the theme aesthetics
- Test volume levels (shouldn't be too loud or jarring)

## Audio Controls

Users can control the audio through the settings menu:
- **Music Volume** - Controls BGM volume (0-100%)
- **SFX Volume** - Controls sound effects volume (0-100%)
- **Music Toggle** - Enable/disable background music
- **SFX Toggle** - Enable/disable sound effects
